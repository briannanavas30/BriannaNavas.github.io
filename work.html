// Work

Professional Projects

During my time at Texas A&M, I’ve been able to work on some cool projects. These include programming, game development, and web design. Here’s a showcase of the work I’m proudest of.

// Check-Up

From TamuHack 2022, Check-Up aims to relieve intrusive thoughts caused by OCD and anxiety through providing a centralized checklist. My contributions included product branding, UX design, promotional video, and Devpost. This project placed Top 5 and won best Devpost out of 100+ projects.

Check out the Devpost page >

// Goodnight Light

Goodnight Light is a pixel art horror game which focuses on a young girl and the menacing monster under her bed. This project was intended to explore level-design techniques to effectively communicate narrative to the player.

View the project proposal >

// Secret Santa Sabotage

This single page RPG revolves around the players designing a plan to sabotage Christmas. This project was intended to explore combining a narrative and game balance. Contributions included mechanic design and creating the visuals.
